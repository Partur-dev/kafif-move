# NitroSpoof

Send Nitro emojis as images.

## Features

- Supports emoji escaping
- Can use emojis if they are unavailable due to a lack of boosts (grayed out in
  emoji picker)
- Removes unavailable emoji tint in emoji picker
- Doesn't spoof premiumType
